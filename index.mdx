FreeRTOS is a market-leading real-time operating system (RTOS) for microcontrollers and small microprocessors. It provides the kernel primitives you need — task scheduling, queues, semaphores, timers — plus a rich ecosystem of libraries for networking, security, and AWS IoT connectivity.

## Get started

<Steps>
  <Step title="Clone the repository">
    Clone FreeRTOS with all submodules to get the kernel and all companion libraries:
    ```bash
    git clone https://github.com/FreeRTOS/FreeRTOS.git --recurse-submodules
    ```
  </Step>
  <Step title="Choose your port">
    Browse `FreeRTOS/Demo/` for a pre-configured project matching your hardware (ARM Cortex-M, RISC-V, AVR, PIC, and many more) and toolchain (GCC, IAR, Keil, MSVC).
  </Step>
  <Step title="Configure the kernel">
    Create a `FreeRTOSConfig.h` file in your project to set heap size, tick rate, task priorities, and enable optional kernel features.
  </Step>
  <Step title="Write your first task">
    Define your application logic as FreeRTOS tasks and start the scheduler:
    ```c
    void vMyTask(void *pvParameters) {
        for (;;) {
            /* Task body */
            vTaskDelay(pdMS_TO_TICKS(1000));
        }
    }

    int main(void) {
        xTaskCreate(vMyTask, "MyTask", 128, NULL, 1, NULL);
        vTaskStartScheduler();
        /* Never reached */
        return 0;
    }
    ```
  </Step>
</Steps>
