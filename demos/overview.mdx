The FreeRTOS distribution includes over 200 demo projects covering a wide range of hardware targets and toolchains. Each demo is a complete, buildable project that demonstrates the kernel and companion libraries running on real hardware or a simulator.

## Demo directories

| Directory | Contents |
|-----------|----------|
| `FreeRTOS/Demo/` | Kernel demos for 40+ processor families |
| `FreeRTOS-Plus/Demo/` | FreeRTOS+ library demos (TCP, CLI, MQTT, HTTP, AWS IoT) |

## Kernel demo categories

### ARM Cortex-M
Demos for Cortex-M0 through M7, including MPU variants, across IAR, Keil, and GCC toolchains:

- `CORTEX_M4F_STM32F407ZG-SK/` — STM32F407 Discovery (GCC + Keil)
- `CORTEX_M7_STM32F7_STM32756G-EVAL_IAR_Keil/` — STM32F7 (IAR + Keil)
- `CORTEX_M0+_Atmel_SAMD20_XPlained/` — Atmel SAMD20
- `CORTEX_MPU_M33F_NXP_LPC55S69_MCUXpresso/` — NXP LPC55S69 with TrustZone

### ARM Cortex-A/R
- `CORTEX_A9_Zynq_ZC702/` — Xilinx Zynq-7000
- `CORTEX_A53_64-bit_UltraScale_MPSoC/` — Xilinx UltraScale+ (64-bit)
- `CORTEX_R4_RM48_TMS570_CCS5/` — TI Hercules RM48

### RISC-V
- `RISC-V_RV32_SiFive_HiFive1-RevB_FreedomStudio/` — SiFive HiFive1 Rev B
- `RISC-V_RV64_PolarFire_SoftConsole/` — Microchip PolarFire SoC

### Microchip / Atmel
- `PIC32MZ_MPLAB/` — PIC32MZ (MPLAB X)
- `AVR_ATmega328PB_Xplained_mini_GCC/` — ATmega328PB
- `RX700_RX72N_EnvisionKit_GCC_e2studio/` — Renesas RX72N

### Other architectures
MSP430, MicroBlaze, NiosII, TriCore, Tensilica Xtensa, and more — see `FreeRTOS/Demo/` for the full list.
