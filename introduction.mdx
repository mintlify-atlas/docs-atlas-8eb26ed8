FreeRTOS is the world's leading real-time operating system (RTOS) for microcontrollers and small microprocessors. It is professionally developed, strictly quality controlled, robust, and free to embed in commercial products without any requirement to expose proprietary source code.

## What is FreeRTOS?

FreeRTOS provides the core primitives every embedded application needs:

- **Task scheduling** — preemptive, cooperative, or hybrid
- **Inter-task communication** — queues, semaphores, mutexes, event groups, stream buffers
- **Memory management** — five heap implementations from fully static to coalescing allocators
- **Software timers** — one-shot and auto-reload timers running in timer daemon task
- **MPU support** — memory protection unit integration for Cortex-M and ARM cores

## Repository structure

```
FreeRTOS/
├── FreeRTOS/
│   ├── Source/          # Kernel source (Git submodule)
│   ├── Demo/            # 200+ hardware demo projects
│   └── Test/            # Kernel tests and CBMC proofs
└── FreeRTOS-Plus/
    ├── Source/          # TCP, CLI, MQTT, HTTP, PKCS11, AWS libraries
    ├── Demo/            # FreeRTOS+ demo projects
    └── ThirdParty/      # wolfSSL, mbedTLS, tinycbor
```

## Long-term support

FreeRTOS publishes LTS releases on a regular cadence. The 202406-LTS release covers the Kernel, FreeRTOS+TCP, and all core libraries. LTS releases receive security patches and bug fixes for an extended period, making them suitable for production deployments.
